Compile flags for gfortran

When compiling Fortran, you cannot be sure, that gfortran sees the code the same way like you. For instance, when you assign 1. to a real(8) variable and don‘t notice, that 1. is only real(4).

But when compiling with some flags from this blog, gfortran will warn you about such issues.
